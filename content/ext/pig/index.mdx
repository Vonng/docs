## Overview

**PIG** (Postgres Install Genius) is an open-source PostgreSQL extension package manager for Linux distributions. It provides a unified command-line interface for managing PostgreSQL extensions, repositories, and Pigsty installations.

## Key Features

- **420+ Extensions**: Access to a comprehensive catalog of PostgreSQL extensions
- **Multi-Platform Support**: Compatible with RHEL, Rocky, CentOS, Debian, and Ubuntu
- **Multi-Architecture**: Works on both x86_64 and ARM64 systems
- **PostgreSQL Version Support**: Supports PostgreSQL versions 13-17
- **Easy Installation**: Simple one-command installation process
- **Integration**: Works with system package managers (YUM/APT)

## Installation

Get started with PIG in seconds:

```bash
curl -fsSL https://repo.pigsty.io/pig | bash
```

This installs the latest PIG package and sets up the necessary repositories for managing PostgreSQL extensions.
