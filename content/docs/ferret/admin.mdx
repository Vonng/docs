## Create FerretDB Cluster

After [defining](https://pigsty.io/docs/ferret/config/#configuration) the FerretDB cluster in the [inventory](https://pigsty.io/docs/setup/config/), you can install it with:

```bash
./mongo.yml -l ferret   # Install MongoDB/FerretDB on ferret group
```

Since FerretDB uses PostgreSQL as its underlying storage, running this playbook multiple times is generally safe.

------

## Remove FerretDB Cluster

To remove a Mongo/FerretDB cluster, run the `mongo_purge` subtask of [`mongo.yml`](https://pigsty.io/docs/ferret/admin/#mongoyml) playbook with the `mongo_purge` parameter:

```bash
./mongo.yml -e mongo_purge=true -t mongo_purge
```
