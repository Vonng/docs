## Installation

Prepare a Linux node that runs any compatible Distros, then install Pigsty with:

```bash title="install"
curl -fsSL https://repo.pigsty.io/get | bash;
cd ~/pigsty; ./bootstrap; ./configure; ./install.yml;
```
