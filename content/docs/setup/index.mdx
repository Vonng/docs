## Quick Start

[Prepare](/docs/setup/prepare) a [ssh-accessible](/docs/setup/prepare#accessibility) [node](/docs/setup/prepare#node) with fresh installed [Compatible Linux OS](/docs/setup/linux),
 run as user with nopass [`ssh`](/docs/setup/prepare#ssh) and [`sudo`](/docs/setup/prepare#admin-user) privileges:

<Steps>
<Step>

[**Download**](/docs/setup/install#download) pigsty with:

```bash
curl -fsSL https://repo.pigsty.io/get | bash; cd ~/pigsty;
```

</Step>
<Step>
[Bootstrap](/docs/setup/offline#bootstrap) pigsty with required dependencies (`ansible`):

```bash
./bootstrap
```
</Step>

<Step>
[**Configure**](/docs/setup/install#configure) the [`pigsty.yml`](/docs/setup/config) inventory file according your need and environment.

```bash
./configure
```

</Step>
<Step>

[**Install**](/docs/setup/install#install) everything accoring to your config:

```bash
./install.yml
```
</Step>


</Steps>


Pigsty Single-Node [**Installation**](/docs/setup/install) Ready! 
Check port `5432` for [postgres](/docs/pgsql) [service](/docs/pgsql/service), port `3000` for grafana dashboards (userpass: `admin` / `pigsty`).
